# 后台使用指南

> 适用版本：v1.1.0  
> 适用对象：网站管理员、运营、内容编辑、销售跟单人员  
> 配套文档：`Documents/系统架构文档.md`、`Documents/网站模板开发指南.md`

---

## 1. 开始之前

后台登录地址通常为：

```text
/admin/login
```

首次安装后的默认账号信息请以项目部署说明为准。登录后建议第一时间完成以下操作：

1. 修改默认管理员密码。
2. 在「系统设置 → 基础设置」中完善网站名称、Logo、Favicon 和 SEO 信息。
3. 在「系统设置 → 联系方式」中补全邮箱、电话、地址、WhatsApp 等联系资料。
4. 在「外观区块 → 网站模版」中确认当前启用主题是否正确。

---

## 2. 后台导航说明

后台顶部主导航分为以下几组：

| 菜单 | 作用 |
|------|------|
| `仪表盘` | 查看产品、内容、询盘、留言等核心统计数据 |
| `收件箱` | 处理客户留言和产品询单 |
| `产品中心` | 管理产品及产品分类 |
| `内容管理` | 管理文章、案例、页面和媒体库 |
| `员工管理` | 创建后台账号并分配权限，仅管理员可见 |
| `外观区块` | 管理网站模版、模板区块、菜单、轮播图，仅管理员可见 |
| `系统设置` | 管理公司资料、翻译、展示图片、自定义代码和程序更新 |

不同角色看到的菜单会不同：

- `管理员`：拥有全部功能。
- `普通员工`：只会看到被授权的模块。

## 3. 仪表盘怎么用

登录后台后默认进入「仪表盘」。

你可以在这里快速查看：

- 产品总数、已上架数量
- 案例、文章、页面数量
- 留言与询单数量
- 待处理询单数量
- 今日、本周、本月互动情况

建议每天先查看仪表盘，再进入对应模块处理内容或客户消息。

## 4. 产品中心

### 4.1 新增产品

进入「产品中心 → 产品列表」，点击“添加产品”。

一个完整的产品通常包含：

1. `产品标题`
2. `产品描述`
3. `产品图片`
4. `横幅图`
5. `产品摘要`
6. `产品分类`
7. `标签 / 供应商 / 产品类型`
8. `SEO 标题 / SEO 描述`

### 4.2 产品别名（Slug）规则

产品编辑页中的 `别名 (Slug)` 用于生成前台网址，例如：

```text
/product/stainless-steel-pipe
```

系统会自动执行以下规则：

- 只允许 `小写字母`、`数字` 和 `-`
- 自动转为小写
- 自动过滤空格、中文和其他特殊字符
- 留空时会根据标题自动生成

建议使用简短、语义清晰的英文标识，例如：

- `stainless-steel-pipe`
- `industrial-valve`
- `custom-cnc-parts`

### 4.3 产品分类

进入「产品中心 → 产品分类」可新建和维护分类。

适合先建立顶级分类，再逐步细分子类，例如：

- `Steel Products`
- `Machinery Parts`
- `Packaging Materials`

如果某个产品没有分类，前台筛选体验会变差，建议发布前先整理分类结构。

## 5. 内容管理

内容管理包含四块：文章、案例、页面、媒体库。

### 5.1 文章管理

适合发布：

- 公司新闻
- 行业资讯
- 采购指南
- 展会动态

建议每篇文章填写：

- 标题
- 摘要
- 分类
- 正文
- SEO 标题
- SEO 描述

### 5.2 案例管理

适合展示：

- 客户合作项目
- 定制案例
- 出口项目
- 工厂交付能力

案例通常建议带封面图，并在标题中体现行业和结果。

### 5.3 页面管理

适合维护相对固定的单页内容，例如：

- 关于我们
- 服务介绍
- 常见问题
- 招商合作

页面同样有 `Slug`，规则与产品一致。页面发布后通常会通过：

```text
/page/about-us
```

访问。

### 5.4 媒体库

进入「内容管理 → 媒体库」可以：

- 上传图片和视频
- 新建文件夹
- 搜索文件
- 按类型筛选
- 删除文件

在其他编辑页面里看到“选择图片”“从媒体库选择”等按钮时，点击后会弹出统一媒体库窗口。  
建议按用途建立文件夹，例如：

- `products`
- `banners`
- `certificates`
- `company`
- `blog`

这样后续维护更方便。

## 6. 收件箱

收件箱分为 `联系留言` 和 `询单管理` 两部分。

### 6.1 联系留言

来源通常是前台联系表单。你可以：

- 查看客户姓名、邮箱、电话、公司
- 查看留言内容
- 删除无效留言

### 6.2 询单管理

询单通常来自产品详情页或询盘表单，信息比普通留言更完整，常见字段包括：

- 客户姓名
- 公司
- 邮箱 / 电话
- 需求产品
- 数量
- 询单内容
- 来源页面
- IP 地址

支持的状态流转为：

- `待处理`
- `已联系`
- `已报价`
- `已关闭`

推荐处理流程：

1. 新询单先标记或保留为 `待处理`
2. 首次联系客户后改为 `已联系`
3. 发出正式报价后改为 `已报价`
4. 成交结束或明确不再跟进时改为 `已关闭`

如需交给 Excel 或 CRM 系统处理，可使用“导出 CSV”。

## 7. 外观区块

只有管理员可以使用这一组功能。

### 7.1 网站模版

进入「外观区块 → 网站模版」后，系统会自动读取 `/themes` 目录下的主题。

你可以在这里完成：

1. 查看已安装主题
2. 查看主题封面图
3. 上传 zip 主题包
4. 启用指定主题

后台会优先读取主题目录中的 `screenshot.jpg` 作为封面。  
如果主题没有 `screenshot.jpg`，页面会显示“无预览图片”。

#### 主题上传要求

上传的 zip 主题包必须满足以下规则：

1. 压缩包里只能有一套主题
2. 主题根目录必须包含 `style.css`
3. `style.css` 顶部必须包含主题元数据，至少要有 `Theme Name`
4. 系统要求的模板文件必须齐全

如果不符合，后台会提示“此文件不符合网站主题的要求”。

### 7.2 模板区块

进入「外观区块 → 模板区块」可以不改代码直接修改页面文案和图片。

当前页面已经按所属页面分组展示区块，常见分组包括：

- `首页`
- `关于我们页面`
- `联系我们页面`
- `全站通用`
- `品牌与样式`

在这里你可以调整：

- 标题
- 描述文字
- 按钮文案
- 图片
- 图标
- 颜色

修改后点击“保存全部区块”即可生效。  
如果某个区块改乱了，可以勾选“重置此区块为模板默认值”。

### 7.3 菜单管理

进入「外观区块 → 菜单管理」可创建导航菜单。

菜单支持：

- 多个菜单
- 多级菜单
- 拖拽排序
- 自定义链接
- 新窗口打开

常见菜单标识符建议：

- `main-nav`：顶部主导航
- `footer`：页脚快捷链接

菜单 `Slug` 规则与其他内容一致：

- 仅支持小写字母、数字和 `-`
- 保存时自动转小写
- 自动过滤非法字符

### 7.4 轮播图

进入「外观区块 → 轮播图」可管理首页或其他页面的轮播内容。

每个轮播图通常包含：

- 轮播图名称
- 标识符
- 描述
- 启用状态
- 多张轮播图片
- 每张图对应的标题、说明、按钮、跳转链接

可以拖拽图片调整顺序。  
建议轮播图图片尺寸保持统一，避免前台跳动。

## 8. 系统设置

系统设置用于维护全站基础资料，主要包含以下标签页：

| 页面 | 用途 |
|------|------|
| `基础设置` | 网站名称、标语、启用主题、SEO、Logo、Favicon、分享图 |
| `公司简介` | 主营产品、公司介绍、员工人数、厂房面积、资质等 |
| `贸易能力` | 出口市场、生产能力、交付能力等 |
| `公司展示` | 公司图片、工厂图、资质图等展示素材 |
| `联系方式` | 电话、邮箱、地址、社交账号、WhatsApp 等 |
| `翻译设置` | 前台 Google Translate 显示相关配置 |
| `自定义代码` | 自定义 head/footer 代码 |
| `程序更新` | 检查版本、安装更新、查看备份和迁移状态 |

建议网站上线前至少完整检查一次 `基础设置`、`公司简介`、`联系方式` 三个页面。

## 9. 员工与权限

进入「员工管理」可以创建后台账号。

可配置内容包括：

- 用户名
- 显示名称
- 密码
- 角色
- 访问权限

### 9.1 角色区别

- `管理员`：拥有全部权限，包括外观区块和员工管理
- `普通员工`：只能访问被勾选授权的模块

### 9.2 权限建议

常见分工方式：

- 运营：开放 `内容管理`、`产品管理`
- 销售：开放 `询单/留言`
- 技术或管理员：开放 `系统设置`、`外观区块`

遵循“够用即可”的原则，不建议给所有员工都开管理员权限。

## 10. 个人资料与账号安全

点击右上角头像菜单，可以进入：

- `个人资料`
- `访问网站`
- `退出登录`

建议定期修改密码，并避免多人共用同一个管理员账号。

## 11. 推荐日常操作流程

如果你是网站日常维护人员，推荐按下面顺序工作：

1. 登录后台查看仪表盘
2. 进入收件箱处理新留言和询单
3. 更新产品、文章或案例内容
4. 在媒体库整理新增图片
5. 如需改首页文案，进入模板区块修改
6. 如需改导航或轮播，进入外观区块对应页面
7. 定期检查系统设置中的联系信息是否仍然准确

## 12. 常见问题

### Q1：为什么上传主题后不能启用？

通常是以下原因：

- 缺少必需模板文件
- `style.css` 不存在
- `style.css` 没有 `Theme Name`
- zip 包里包含了多个主题目录

### Q2：为什么主题没有封面图？

因为主题目录里没有 `screenshot.jpg`。  
补上该文件后刷新「网站模版」页面即可显示。

### Q3：为什么别名会被自动改成小写？

这是系统的统一规则，用来保证前台链接稳定、规范。  
后台所有用户输入的 `Slug` 都只允许：

- 小写字母
- 数字
- 连字符 `-`

### Q4：模板区块保存后前台没变化怎么办？

建议依次检查：

1. 当前启用的主题是否正确
2. 修改的区块是否属于当前主题
3. 是否点击了“保存全部区块”
4. 是否误勾选了“重置此区块为模板默认值”
